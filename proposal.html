Project Proposal

Problem/Data Domain

Overview of General Topic

Hustle culture is a big part of society and this pressures many people to set big goals far into the future, which makes many work hard to achieve those goals that they do not have time to think about what is happening in the present. Although it can lead to success, hustle culture can also lead to an unideal work-life balance that can ruin mental and physical well-being.

Information Technology Problem

An information technology problem that arises from hustle culture is that planner apps are geared toward setting goals and achieving them through tasks, but the main focus is the goal that will get completed sometime in the future. This means that people are more focused on their goals in the future rather than the events that are happening in the present.

Previous/Existing Solution

Some examples of planner apps are any.do and bordio. These two solutions have the problems that were mentioned in the previous section where they put all the focus on setting goals in the future.

App Description

Who are the Users?

Users can be of any age. They are people who want to take a break from having to deal with the big things in life and to instead take some time to live in the present and think about something positive. Since this app needs to be accessed through a computer, people who do not have access to computers would not be able to use this app.

What will the Users Do?

Four Main Components User will be Interacting with:

  • Checklist of things the user wants to complete that day
  • Moodboard with pictures the user wishes to place onto the bulletin board.
  • Diary page where the user can write down any notable events or thoughts that occurred on that day.
  • Sticky note where the user can write a positive thing that happened that day.
General Components of the App:
  • The user will be able to create their daily planner at the start of each day.
  • At 12:00 am everyday, the daily planner will be archived and users can access it anytime they want.
  • The user will have to login/sign-up by submitting a form with their username and password.
  • The user can pick their favorite daily planners and put them into a separate archive.

What Kind of Information will the Users be Viewing?

The user will be viewing the data they put into the daily planner, so all the data for the website is curated by the user. For example, if the user fills out the checklist, they will be able to see the items in that checklist for the day. Another example is if the user wants to see their archived daily planners, they can click on a button that will allow them to see all the daily planners they have completed.

How will Using this App Solve the Chosen Problem?

Since this app focuses only on a single day, the user can focus on what happens in their day and to live more in the moment. The sticky note that prompts the user to write down something positive that happened encourages the user to focus on the positive aspects of their day and to help them find joy in the little things.
